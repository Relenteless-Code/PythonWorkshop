# 🐍 Python Workshop (13 Days)

Welcome to the **Python Workshop of 13 Days**!  
This repository contains all the materials, examples, and exercises covered during the workshop.  
It is designed for beginners and intermediate learners who want to strengthen their Python skills step by step.

---

## 📅 Workshop Schedule

| Day | Topic |
|-----|-------|
| 1   | Introduction to Python, Installation, First Program |
| 2   | Variables, Data Types, and Input/Output |
| 3   | Operators (Arithmetic, Relational, Logical, Bitwise) |
| 4   | Control Flow (if, else, loops) |
| 5   | Functions and Scope |
| 6   | Strings and String Methods |
| 7   | Lists, Tuples, and Sets |
| 8   | Dictionaries and Collections |
| 9   | File Handling (read/write) |
| 10  | Exception Handling |
| 11  | Modules and Packages |
| 12  | Object-Oriented Programming (Classes, Objects, Inheritance) |
| 13  | Final Project + Revision |
